# Get HGNC cache directory

Returns the path to the directory where HGNC data is cached. Creates the
directory if it doesn't exist.

## Usage

``` r
get_hgnc_cache_dir()
```

## Value

Character string with the path to the cache directory

## Examples

``` r
if (FALSE) { # \dontrun{
get_hgnc_cache_dir()
} # }
```
