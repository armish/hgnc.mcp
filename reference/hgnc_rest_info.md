# Get HGNC REST API Information (Cached)

Cached version of hgnc_rest_info_uncached(). Uses memoise to cache
results.

## Usage

``` r
hgnc_rest_info()
```

## Details

This is a memoised wrapper around hgnc_rest_info_uncached(). For
complete documentation, see hgnc_rest_info_uncached() or run:
roxygen2::roxygenise()
